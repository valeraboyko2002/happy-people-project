# World Happiness Report: Анализ и Прогнозирование Уровня Счастья

## О проекте

Этот проект посвящен анализу данных из **World Happiness Report** - ежегодного исследования уровня счастья в странах мира. Цель проекта - выявить ключевые факторы, влияющие на уровень счастья, и построить модель для его прогнозирования.



## Цели проекта

1. **Аналитическая**:
   - Выявить ключевые факторы, влияющие на уровень счастья
   - Проанализировать региональные различия
   - Обнаружить выбросы и аномалии

2. **Машинное обучение**:
   - Построить модель прогнозирования уровня счастья
   - Сравнить различные алгоритмы
   - Достичь высокой точности прогнозирования

## Ключевые показатели

- **Количество стран**: 140
- **Количество признаков**: 6 основных факторов + 3 мета-признака
- **Целевая переменная**: Ladder Score (оценка счастья от 1.7 до 7.7)
- **Лучшая модель**: Ridge Regression с R² = 0.8105

## Основные факторы счастья

По корреляции с уровнем счастья:

1. **Социальная поддержка** (0.814) 
2. **ВВП на душу населения** (0.769) 
3. **Ожидаемая продолжительность здоровой жизни** (0.759) 
4. **Свобода жизненного выбора** (0.644) 
5. **Восприятие коррупции** (0.452) 
6. **Щедрость** (0.130)

## Ключевые инсайты

### 1. **Линейность данных**
- Все основные факторы имеют сильную линейную связь с уровнем счастья
- Это объясняет превосходство линейных моделей над деревьями

### 2. **Региональные особенности**
- **Северная Европа**: высочайший уровень счастья
- **Южная Азия**: самый низкий уровень счастья
- **Афганистан**: экстремальный случай по всем показателям

### 3. **Выбросы как реальные кейсы**
- **Дания/Финляндия**: почти нулевое восприятие коррупции
- **Афганистан**: нулевая социальная поддержка
- **Индонезия/Мьянма**: высокая щедрость
